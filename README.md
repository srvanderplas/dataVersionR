# dataVersionR

[![Travis build

[![Coverage

The goal of dataVersionR is to provide methods for saving a hashed
record of the inputs to a function along with the output of the
function.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("srvanderplas/dataVersionR")
```
